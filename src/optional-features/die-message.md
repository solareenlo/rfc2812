# 4.3 Die メッセージ

```
   Command: DIE
Parameters: None
```

オペレータは、DIE コマンドを使用してサーバをシャットダウンすることができます。任意の人がオペレータとしてサーバに接続し、このコマンドを実行することはリスクとみなされる可能性があるため、このメッセージはオプションです。

DIE コマンドは、常に送信側クライアントが接続しているサーバで完全に処理されなければならず、接続している他のサーバに渡してはなりません。

数値返信：

```
        ERR_NOPRIVILEGES
```

例：

```
DIE
    ; パラメータは必要ありません。
```
